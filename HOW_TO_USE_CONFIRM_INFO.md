# 如何使用 Confirm Information 功能

## 🎯 快速开始

### **步骤 1：打开应用**
启动 Demo Sample 应用，进入 Home Page。

### **步骤 2：进入 Test Figma**
点击紫色的 **"Test Figma"** 按钮。

### **步骤 3：选择 Confirm Information**
在 Test Figma Page，你会看到两个按钮：
- 🟡 **ID Scanner** (金色) - 实时相机扫描
- 🔵 **Confirm Information** (浅蓝色) - 文档上传 UI ⭐

点击蓝色的 **"Confirm Information"** 按钮。

### **步骤 4：浏览文档上传页面**
你会看到一个完整的文档上传界面：
- 📋 "Upload Supporting Document" 标题
- 📄 虚线边框的上传区域
- 📷 "Open Camera & Take Photo" 按钮
- ✅ "Next" 按钮
- 📱 底部导航栏

### **步骤 5：探索 UI**
- 点击虚线上传区域 - **无反应**（装饰性）
- 点击 "Open Camera" 按钮 - **无反应**（装饰性）
- 点击 "Next" 按钮 - **无反应**（装饰性）
- 点击底部导航图标 - **无反应**（装饰性）

### **步骤 6：返回**
- 点击左上角返回按钮 ← 返回 Test Figma Page
- 或点击右上角关闭按钮 ✕ 返回 Test Figma Page

## ❓ 常见问题

### **Q: 为什么点击按钮没有反应？**
**A:** 这是设计需求。这是一个**纯 UI 展示功能**，用于展示界面设计，不包含实际的上传功能。

### **Q: 可以实际上传文件吗？**
**A:** 不可以。当前版本仅为 UI 展示。如需实际功能，可在后续开发中添加。

### **Q: 与 ID Scanner 有什么关系？**
**A:** 两者都是 Test Figma 功能集的一部分，但功能不同：
- ID Scanner = 完整功能演示
- Confirm Information = UI 设计演示

### **Q: 底部导航图标有用吗？**
**A:** 不能。底部导航是装饰性的，用于展示导航栏设计。

### **Q: 这会影响其他功能吗？**
**A:** 不会。这是完全独立的功能，不影响 Scan ID、PDF 生成、表单填充等任何现有功能。
